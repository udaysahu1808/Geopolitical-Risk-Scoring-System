# Geopolitical-Risk-Scoring-System
It is an end to end project using SQL, Python, Machine Learning and Power BI. Connected Python with SQL for automated data extraction and preprocessing, built an ML-based risk scoring model to assess country-level geopolitical threats and designed interactive Power BI dashboards for real time risk monitoring, trend analysis and decision making.
